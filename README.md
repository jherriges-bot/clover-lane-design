# Clover Lane Design App

Deployable Vite + React project for:
- crew timeclock
- manual time entry
- employee / job / labor category setup
- estimate vs actual reporting
- Google Sheets sync through Apps Script
- installable phone/desktop web app

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to GitHub and import into Vercel.
